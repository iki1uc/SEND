# SEND · Transferstation · Push/Send

SEND ist die Ausgabestation des iki1uc-Systems.
Sie sendet Werte weiter, die HOLD festhält.

──────────────────────────────────────────────
## Achsen

- state – READY / EMPTY / BLOCKED
- push – Senden
- sent – gesendeter Wert

──────────────────────────────────────────────
